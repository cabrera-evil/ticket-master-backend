# La Cuponera SV API Docs

La documentacion API oficial ahora se genera automaticamente con Scramble (OpenAPI 3.1).

## Endpoints de documentacion

- UI interactiva: `/api/docs`
- OpenAPI JSON: `/api/docs.json`

Ejemplo local:

- `http://localhost:8080/api/docs`
- `http://localhost:8080/api/docs.json`

## Notas

- La fuente de verdad para contratos API es la documentacion generada desde rutas, `FormRequest` y recursos.
- Este archivo queda como punto de entrada y referencia rapida.
